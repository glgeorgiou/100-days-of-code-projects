What is Python

Python is a high-level general-purpose programming language

  • high-level: Python is a more abstract language compared to C. This adds expressiveness to Python and offers power to programmers to write more flexible and efficient code. The downsize might be that Python code is slower in occasions, but many optimization methods exist to deal with this issue.
  • eneral-purpose: Python is not bound to any specific programming purpose (unlike R which is specifically 'statistics-oriented') nor programming paradigm (unlike Java which imposes object-oriented programming techniques). This feature makes Python a powerful tool for a variety of applications in various scientific domains and also in education. The interested developer will find numerous domain-specific packages to start writing specialized code for the domain of interest. The teacher has a versatile tool to advance students' computational thinking.
